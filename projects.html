hex_tiles.js

A widget for drawing a configurable map of hex tiles. Each tile is represented in the data model as an object with various properties. Tiles can be colored, dragged, and selected. Mostly a base for another potential project.

Controls:
A = drag mode
S = select mode
D = draw mode
F (hold) = cycle color of each tile
R = reset
Z/X = zoom in/out
